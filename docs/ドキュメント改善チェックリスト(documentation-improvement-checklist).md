### Task 1.2: パス参照修正
**作業内容**: `Linux-ExHDD` → `LinuxHDD` への置換
**期間**: 2日（12/24-25）
**担当**: Technical Writer
**優先度**: 🔴 高

**修正対象ファイル**:
```bash
# 修正前の確認
grep -r "Linux-ExHDD" docs/
```

**修正方法**:
```bash
# ファイル一覧の取得
FILES=$(grep -rl "Linux-ExHDD" docs/ 2>/dev/null)

# ファイルごとに確認・修正
for file in $FILES; do
    echo "修正ファイル: $file"
    sed -i 's|Linux-ExHDD|LinuxHDD|g' "$file"
done
```

**修正確認**:
```bash
# 修正後の確認（0件になるはず）
grep -r "Linux-ExHDD" docs/ | wc -l
```
