# 画面設計書 (UI Design)

## 1. 概要

本ドキュメントは、Personal Recipe Intelligence (PRI) の画面設計を定義する。

## 2. デザイン原則

### 2.1 基本方針
- **シンプル**: 必要最小限の要素
- **高速**: 軽量なコンポーネント
- **直感的**: 分かりやすいUI
- **一貫性**: 統一されたデザイン

## 4. コンポーネント仕様

### 4.1 RecipeCard

```svelte
<script>
  export let recipe;
</script>

<div class="recipe-card">
  <img src={recipe.image} alt={recipe.title} />
  <div class="content">
    <h3>{recipe.title}</h3>
    <div class="meta">
      <span class="time">{recipe.cook_time}分</span>
      <span class="servings">{recipe.servings}人分</span>
    </div>
    <div class="tags">
      {#each recipe.tags as tag}
        <span class="tag">{tag}</span>
      {/each}
    </div>
  </div>
</div>
```

### 4.2 SearchBox

```svelte
<script>
  export let value = '';
  export let placeholder = 'レシピを検索...';

  function handleSearch() {
    dispatch('search', { query: value });
  }
</script>

<div class="search-box">
  <input
    type="text"
    bind:value
    {placeholder}
    on:keydown={(e) => e.key === 'Enter' && handleSearch()}
  />
  <button on:click={handleSearch}>検索</button>
</div>
```

### 4.3 TagChip

```svelte
<script>
  export let tag;
  export let removable = false;
</script>

<span class="tag-chip">
  {tag.name}
  {#if removable}
    <button class="remove" on:click={() => dispatch('remove')}>×</button>
  {/if}
</span>
```
