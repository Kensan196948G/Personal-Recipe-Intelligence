## 同義語の活用

システムは以下の同義語を自動的に認識します：

### 食材
- `たまねぎ` / `タマネギ` → `玉ねぎ`
- `とり肉` / `チキン` → `鶏肉`
- `ぶた肉` / `ポーク` → `豚肉`
- `にんじん` / `キャロット` → `人参`
- `たまご` / `エッグ` → `卵`

### 形容詞
- `かんたん` / `カンタン` → `簡単`
- `健康的` → `ヘルシー`

## ヒントとコツ

### 1. 具体的に書く
❌ `料理` → 🆗 `簡単な鶏肉料理`

### 2. 否定は明確に
❌ `辛い` → 🆗 `辛くない`

### 3. 複数条件を組み合わせる
🆗 `和食でヘルシーで魚なし`

### 4. 自然な日本語で
🆗 `時短でできる簡単なパスタ`

### 5. サジェストを活用
入力途中で候補が表示されるので活用しましょう

## トラブルシューティング

### 結果が多すぎる場合
→ 条件を追加してください
例: `鶏肉` → `鶏肉で簡単で和食`

### 結果が少なすぎる場合
→ 条件を減らしてください
例: `和食でヘルシーで魚なしで簡単で時短` → `和食でヘルシー`

### 期待と違う結果が出る場合
→ クエリ解析結果を確認して調整してください
例: 解析結果で意図しない単語が抽出されていないか確認
